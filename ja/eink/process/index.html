Operations // DOC NO. RL-200-A

私たちのプロセス

私たちは人々に私たちを探してもらうよう求めません。素材がすでに行き着く場所、つまりそれが生まれた事業所で素材を受け取ります。私たちの回収は地域の持ち込みパートナーのネットワークで成り立っており、各パートナーは顧客が生み出す特定の廃棄物の流れに合わせたビンを設置しています。

SEC.01

持ち込みモデル

販売の場で回収する

私たちは地元の事業所と提携し、もともと彼らの購入品の一部だったリサイクル可能素材の持ち込み拠点とします。

SPEC / COLLECTION MODELRL-200
  • モデル:パートナー事業所に設置する持ち込みビン
  • 起点:素材はそれ自身の販売の場へ戻る
  • 範囲:ローカル。ここで回収し、ここで処理し、ここで使う
  • 分別:受け入れ時に流れを分けたまま保つ
  • パートナーの費用:なし。ビンは私たちが提供・管理する

事業所が販売する素材は、しばしば廃棄物としてその同じ扉から戻ってきます:端材、使用済みのセル、寿命を終えたデバイスです。別個の回収システムを構築するのではなく、私たちはその自然な返送地点にビンを置き、ループが始まる場所で閉じます。

顧客にとって、持ち込みは最初の購入と同じくらい手軽です。パートナーにとっては、素材を埋め立てから守るという目に見えるコミットメントです。私たちにとっては、クリーンで予測可能な、事前に分別された供給を意味します。

SEC.02

プラスチックの流れ

模型店からのランナー & スプルー

私たちのプラスチック原料はスケールモデルキットから来ます:パーツを切り離したあとに残るランナーとスプルーです。

SPEC / PLASTIC INTAKERL-P02
  • 供給源:ホビー店、模型キット店、ゲーム店
  • 素材:ランナー & スプルー · ポリスチレン / ABS
  • 状態:設計上、クリーンで乾燥し、汚染がない
  • 分別:可能な限り色ごとに分ける
  • 市場:PS/ABS · BC には買い取る業者がなく、最寄りは Ontario2
  • 行き先:粉砕 → 分別 → 乾燥 → 射出成形

すべての模型キット(Gunpla やその仲間)は、プラスチックの格子に保持されたパーツとして届きます。組み立てが終わると、その格子は捨てられます。これは入手可能な廃棄物の流れの中でも最もクリーンなもののひとつです:均一で高品質なポリスチレンと ABS、既知の規格で成形され、フレームごとに分別され、食品や液体による汚染が一切ありません。

ホビー店や模型店は理想的なパートナーです。彼らの顧客はまさにこの端材を安定して生み出しており、多くのビルダーはすでにランナーを捨てずに取っておいています。私たちは店にビンを置きます。素材はまっすぐ私たちの射出成形ループへと流れ込みます。

このような産業由来の樹脂は、家庭回収のプラスチックよりクリーンで均質なため、割増の価値をもちます1。それでも、BC のどのプログラムも事業所からのプラスチックを引き取らず、地元で PS や ABS にお金を払う業者もいません2。そのギャップこそが私たちの原料です。

SEC.03

バッテリーの流れ

修理店・ベイプ店・小売店からのセル

リチウムセルは、デバイスが販売または修理されるあらゆる場所に蓄積します。私たちはそうした事業所と提携してそれらを捕捉します。

SPEC / CELL INTAKERL-H02
  • 供給源:携帯電話 / IT 修理店
  • 供給源:電子タバコ & ベイプ販売店
  • 供給源:電器小売店(例:Best Buy)
  • 素材:携帯電話、使い捨て品、デバイスからのリチウムセル
  • 取り扱い:端子をテープで絶縁し、防火容器で保管
  • 輸送:リチウム危険物 · UN3480 / UN3481、Class 93
  • 行き先:Project HEX による回収 + 再構築

寿命を終えたセルは家庭回収のビンでは危険ですが、適切な手にかかれば価値があります。私たちは毎日それらを扱う事業所を通じて回収し、Project HEX へと送って、試験、選別、そしてモジュラー電源ユニットへの再構築を行います。

3種類のパートナーがこの流れを供給します:

携帯電話 & IT 修理店:バッテリー交換や基板修理の際に取り出されるセルで、それ以外は死蔵または廃棄される。
電子タバコ & ベイプ販売店:使用済みの使い捨て品やモジュールで、それぞれ、まだ寿命を残したまま捨てられるのが通例のリチウムセルを内蔵している。
電器小売店:Best Buy のような大型店で、すでに回収プログラムを運営し、セルを大量に集めている。

BC には現在ベイプの回収制度がありません。生産者資金による制度が2027年10月に向けて法制化されており、計画は2026年12月までに提出予定です。処理業者はまだ指名されておらず、Quebec の稼働中プログラムでも同様です4。捨てられたセルはまた、2025年に US とカナダ全体で 448 件の廃棄物施設火災を引き起こしました5。いまそれらを捕捉することは、安全対策であると同時に一歩先んじることでもあります。

すべてのセルは端子をテープで絶縁し、防火保管
SEC.04

パートナーネットワーク

ビンを設置するのは誰か

各パートナーは、顧客がすでに生み出している流れを受け入れます。

TYPE 01ホビー & 模型店

顧客が組み立てを終え、残った格子を取り分けておく模型キット店やゲーム店。

持ち込み:ランナー & スプルー
TYPE 02携帯電話 & IT 修理

劣化したバッテリーを交換し、取り出したセルを、きれいに処分する経路もないまま溜め込む修理工房。

持ち込み:リチウムセル
TYPE 03ベイプ販売店

それぞれ密封されたリチウムセルを内蔵した、使用済みの使い捨て品を集める電子タバコ店。

持ち込み:使い捨てセル
TYPE 04電器小売店

確立された回収プログラムを持ち、セルを大量に集める Best Buy のような大型店。

持ち込み:各種デバイスのセル
持ち込みパートナーになるこうした流れのいずれかを生み出す店を運営していますか?ビンの設置に費用はかからず、顧客の素材をローカルなループに留められます。設置のご相談はお問い合わせまで。
