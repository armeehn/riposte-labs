Field guide // DOC NO. RL-101-A

リサイクル101

主要なあらゆる素材の流れについての実践ガイド:何がビンに入り、何が入らず、そしてなぜかを解説します。ルールは自治体によって異なります。このページは基本として、地元のプログラムを最終的な判断として扱ってください。

SEC.01

基本

// リサイクルが実際にどう機能するか

リサイクルは製造のサプライチェーンであり、魔法のビンではありません。

ビンに入れたものは何でも回収され、素材回収施設(MRF)で分別され、梱包され、原料としてメーカーに販売されます。梱包物が汚染されていると(食品残渣、誤った素材、絡まった品物)、その価値は下がり、結局埋め立てられることもあります。

だからルールが重要なのです:きれいに、空に、乾かして、ばらばらに。油まみれの容器ひとつが、紙の梱包物ひとかたまり全体を台無しにしかねません。ビニール袋は分別機械を詰まらせます。うまくリサイクルすれば、素材は実際に second life を得ます。当てずっぽうでやれば、たいていは得られません。

そのギャップの規模は大きい:カナダは2022年に 5,152 kt のプラスチックを廃棄し、そのうち新しいペレット、フレーク、繊維へ再処理されたのは約 6.5% にすぎませんでした。1

地元のルールが優先どの自治体も受け入れる素材の組み合わせが異なります。地元のプログラムのリストを一度確認しましょう。5分で済み、何年も当てずっぽうを続けるよりずっと確実です。
SEC.02

プラスチック

// 樹脂コードを知る

三角形の中の数字は樹脂コードであって、リサイクル可能であるという約束ではありません。

一部の樹脂だけが強いリサイクル市場を持ちます。ボトルとジャグ(#1、#2)は確実なもので、たいていのプログラムは #5 のタブも受け入れます。フィルム、袋、発泡材は、家庭回収のビンにはまず入りません。

1PETボトル、瓶:広く受け入れられる
2HDPEジャグ、タブ:広く受け入れられる
3PVCパイプ、ブリスターパック:ほとんど受け入れられない
4LDPEフィルム、袋:店頭持ち込みのみ
5PPタブ、キャップ:多くのプログラムで受け入れ
6PS発泡材、カトラリー:ほとんど受け入れられない
7Other混合樹脂:一般に受け入れられない
ビン:たいてい可
  • 飲料 + 食品のボトルと瓶(#1)
  • 牛乳ジャグ、洗剤ボトル(#2)
  • ヨーグルトのタブ、テイクアウト容器(#5、地元で確認)
  • すすいで、蓋を付けたまま(たいていのプログラム)
ビン:不可
  • ビニール袋 + フィルム(分別機を詰まらせる。店頭持ち込みを使う)
  • あらゆる種類の発泡材 / ポリスチレン
  • ストロー、カトラリー、そしてクレジットカードより小さいもの全般
  • 「堆肥化可能」なバイオプラスチック(まったく別の流れ)
Riposte の視点ポリスチレン(#6)と ABS(#7)、模型キットに使われる樹脂には、お金を払う BC の買い手がおらず、そもそも BC のどのプログラムも事業所からのプラスチックを引き取りません。2 そのクリーンで単一樹脂の素材こそ、私たちのローカルループ射出成形プログラムが存在する理由です:私たちはそれを引き取り、長く使える製品へと成形します。
SEC.03

紙 + 段ボール

// 繊維の流れ

紙の繊維は、きれいで乾いた状態が保たれる限り、擦り切れるまでに5回から7回リサイクルできます。

段ボールは家庭回収で最も価値のある素材のひとつです。正しく分別され、場所を取らないよう、箱は平らにつぶしましょう。濡れた繊維や食品で汚れた繊維はリサイクルできません:油は物理的にパルプ化の工程を台無しにします。

ビン:可
  • 平らにつぶした段ボール箱
  • オフィス用紙、郵便物、新聞、雑誌
  • 板紙(シリアル箱、靴箱)
  • きれいなピザ箱の蓋(油の付いた半分は切り取る)
ビン:不可
  • 油や食品で汚れた紙
  • 使用済みのペーパータオル、ナプキン、ティッシュ
  • ワックスやプラスチックでコーティングされたカップ(たいていのプログラム)
  • ばらのシュレッダー紙(袋入りは分別機を詰まらせる。地元で確認)
SEC.04

ガラス

// 無限にリサイクル可能

容器ガラスは、品質を損なうことなく永遠に溶かして作り直せます。

落とし穴は:容器ガラスに限るということです。窓ガラス、飲用グラス、陶磁器、耐熱ガラス(耐熱容器など)は融点が異なり、新しい容器の一かたまりを台無しにします。自治体によってはガラスを別途回収するか、まったく回収しないところもあります。お住まいの地域を確認してください。

ビン:可
  • ボトル:色を問わず、すすいだもの
  • 食品瓶、蓋を外して(金属の蓋は金属と一緒にリサイクル)
ビン:不可
  • 窓ガラスや鏡のガラス
  • 飲用グラス、陶器、磁器
  • 耐熱調理器具
  • 電球(CFL は有害廃棄物として持ち込み)
SEC.05

金属

// 最も価値の高い流れ

アルミニウムのリサイクルは、鉱石から製造する場合より約 95% 少ないエネルギーで済みます。まさにリサイクルの成功例です。

アルミニウムとスチールの缶は無期限にリサイクルできます。スクラップ金属(鍋、工具、針金)も価値がありますが、それはスクラップ置き場や金属の持ち込み拠点に出すべきものであり、設備を傷める家庭回収のビンには入れないでください。

ビン:可
  • アルミ缶、すすいだもの
  • スチール / ブリキの食品缶
  • きれいなアルミホイル、丸めたもの(地元で確認)
  • 空のエアゾール缶(たいていのプログラム)
ビン:不可
  • スクラップ金属、工具、調理器具(スクラップ置き場へ)
  • 針金、ケーブル、ハンガー(絡まるもの)
  • 中身の残ったエアゾール缶やプロパン缶(危険)
SEC.06

バッテリー + 電子廃棄物

// ビンには決して。絶対に。

バッテリーは、リサイクルであれゴミであれ、いかなる家庭回収のビンにも入れてはいけません。押しつぶされたリチウムセルは、トラックや施設で火災を引き起こします。

どのバッテリーの種類にも適切な持ち込み経路があります:家庭用バッテリー回収拠点、有害廃棄物施設、小売店の回収プログラムです。電子機器は回収可能な金属と有害な部品の両方を含むため、専用の電子廃棄物プログラムに出すべきものです。

使い捨てベイプは最悪の違反者です:プラスチックに密封され、ゴミとして売られるリチウムセルです。そのほとんどは、セルがまだ健全なまま捨てられます。内蔵されたセルは火災の媒介にもなります:捨てられたバッテリーは、2025年に US とカナダ全体で 448 件の廃棄物施設火災を引き起こし、ベイプは名指しされた要因のひとつでした。3 これこそ Project HEX が捕捉するために存在する廃棄物の流れです:私たちはそれらを回収し、セルを取り出し、モジュラーで太陽光を上面に備えた電源ユニットへと組み上げます。

カナダでは回収制度がいまだにまばらです。Quebec は唯一稼働しているベイプ制度を運営しています(2025年開始)。BC には現在ひとつもありませんが、生産者資金による制度が 2027年10月 に向けて法制化されています。4 それまでは、家庭用バッテリーの持ち込み拠点か専門の回収業者が、唯一の適法な経路です。

適切な経路
  • 家庭用バッテリー → 回収拠点(多くの小売店)
  • リチウム / 充電式 → バッテリー回収プログラム
  • 携帯電話、ノートパソコン、ケーブル → 電子廃棄物の持ち込み
  • 使い捨てベイプ → ベイプ回収、または私たちへ送付
決して不可
  • 家庭回収のリサイクルに入れるあらゆるバッテリー
  • 家庭ゴミに入れるあらゆるバッテリー
  • 家庭回収のリサイクルに入れる電子機器
  • テープ絶縁していないばらのリチウムセル(端子をテープで絶縁する)
現在回収中Riposte Laboratories は、セル回収のために使用済みの使い捨てベイプを受け入れています。持ち込みのご相談はお問い合わせまで。
SEC.07

有機物

// 生ゴミ + 庭ゴミ

埋め立て地の生ゴミは酸素なしで分解され、強力な温室効果ガスであるメタンを生み出します。堆肥化すれば、それは土になります。

お住まいの地域にグリーンビン回収があれば、生ゴミや庭の刈りくずに使いましょう。なければ、家庭での堆肥化で果物や野菜くず、コーヒーかす、卵の殻、落ち葉を処理できます。肉、乳製品、油は家庭堆肥には入れないでください。害虫を引き寄せ、うまく分解されません。

堆肥:可
  • 果物 + 野菜くず、コーヒーかす、卵の殻
  • 庭の刈りくず、落ち葉、無地の紙(家庭堆肥)
  • 自治体のグリーンビンでの肉 + 乳製品(地元で確認)
堆肥:不可
  • あらゆる種類のプラスチック(家庭での「堆肥化可能」バイオプラスチックを含む)
  • 裏庭の堆肥の山への肉、乳製品、油
  • ペットの排泄物(病原体)
SEC.08

繊維

// 忘れられた流れ

衣類は決して家庭回収のビンには入れませんが、そのほぼすべては適切な経路を通じて再利用またはリサイクルできます。

着られる衣類は寄付するか再販すべきです。傷んだ繊維にもまだ価値があります:多くの寄付センターや専門の繊維リサイクルプログラムは、擦り切れた品物を受け入れ、断熱材、ウエス、繊維の詰め物へと変えます。衣類を埋め立てることは、素材と、それを作るのにかかったすべてを無駄にします。

適切な経路
  • 着られる → 寄付、再販、修理
  • 傷んだもの → 繊維リサイクルの持ち込み(品物に「rags」と表示)
  • 靴、リネン、カーテン → ほとんどの繊維プログラムが受け入れる
決して不可
  • 家庭回収のリサイクルに入れるあらゆる繊維(絡まるもの)
  • 寄付ビンに入れる濡れた繊維やカビた繊維
SEC.09

黄金律

// 他に何も覚えられないなら

5つのルールが、良いリサイクルの 95% をカバーします。

01きれいに、空に、乾かして

容器をすすぐ。ソースの付いた瓶ひとつが、梱包物ひとかたまり全体を汚染しかねない。

02リサイクル品を袋に入れない

品物はばらのままビンに入れる。袋入りの荷は開けられずに埋め立てられる。

03絡まるものは禁物

ホース、コード、針金、繊維、鎖:これらは分別ラインを止める。

04バッテリーは決してビンに入れない

リサイクルでもゴミでもない。持ち込みのみ。火災は仮定の話ではない。

05迷ったら、調べる

ウィッシュサイクル(願望リサイクル)は埋め立てより害が大きい。地元のリストを確認する。
