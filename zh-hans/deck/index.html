投资者简报 · 2026 · RIPOSTE LABORATORIES INC.

SEC.01 问题所在 // 完好的价值被丢弃

我们丢弃的材料，大部分寿命尚存。

6.5%
加拿大每年废弃的 5,152 kt 塑料中，仅约 6.5% 被再加工为新材料；其余被填埋或焚烧。1
$0 in BC
BC 没有项目收购工业下脚料 PS 或 ABS，也就是每套模型套件中的树脂。最近的付费买家大约在 4,000 km 以东的 Ontario。2
Oct 2027
BC 由生产者出资的电子烟回收已立法但尚未落地；尚未指定任何处理商，即便在 Quebec 已运行的计划中也是如此。4

废弃物流不是一个处置问题。它是被错误定价的库存：已分拣、已浓缩，每天送到一组可预测的门口。

SEC.02 核心论点 // 反击

riposte：格挡之后的反击。

在材料被丢弃的那一刻回收它，将其再造工程为耐用产品，并返还给同一个社区。短闭环，无需远洋运输，价值在本地捕获。

SEC.03 商业模式 // 在销售点收集

无需四处寻觅的供应。

我们在那些产品会变成废弃物的商家处托管投放箱；材料从它自己的前门回流，预先分拣且洁净。像这样的工业下脚料树脂比路边回收的塑料更洁净、更均匀，因此能获得溢价3；我们的原料免费到货，因为如今 BC 没有买家争抢 PS 或 ABS。2

SEC.04 产品线 // 两条物流，一个闭环

两种原料。两条产品线。

Plastic Works
模型套件的 PS/ABS 流道经破碎、分拣，注塑成型为耐用制品；这些正是 BC 无买家收购的树脂，2 就地再造，就地销售。
Project HEX
回收的锂电芯经测试、分级，重建为模块化六边形电力砖：旗舰产品。

Plastic Works 为近期营收和收集网络提供资金；Project HEX 则是风投规模的机会。

SEC.06 愿景 // 一层自充电外皮

六边形密铺。于是我们包覆整座住宅。

每块电力砖都能发电、储电、互联。拼接足够多，阵列便成为建筑的外表面：一座自供电的测地住宅。阵列无缝扩展，包覆复杂曲面，每次以一块回收电力砖递增。废弃物流化为分布式能源基础设施。

SEC.07 为何在此刻 // 顺风齐至

属于回收、分布式电力的时刻。
