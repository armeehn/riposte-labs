运营 // 文件编号 RL-200-A

我们的工艺

我们不要求人们来找我们，而是在材料本已汇聚之处、在它所来自的商家那里与之相遇。我们的收集依托一个本地投放合作网络，每家合作伙伴都设有一个专门对应其顾客所产生废弃物流的收集箱。

SEC.01

投放模式

在销售点收集

我们与本地商家合作，将其设为投放点，收集那些原本就是其顾客某次购买一部分的可回收材料。

SPEC / 收集模式RL-200
  • 模式：在合作商家托管投放箱
  • 源头：材料回到它自己的销售点
  • 半径：本地。就地收集，就地加工，就地使用
  • 分拣：各物流在收料时即分开
  • 合作成本：无；收集箱由我们提供并维护

一家商家所售出的材料，往往会以废弃物的形式从它自己的门口回流：一段边角料、一枚用尽的电芯、一台报废设备。与其另建一套收集系统，我们不如在这个自然的回流点放置一个收集箱，在闭环开始之处将其闭合。

对顾客而言，投放与当初的购买同样便捷。对合作伙伴而言，这是让材料远离填埋场的可见承诺。对我们而言，这意味着洁净、可预测、预先分拣的稳定供应。

SEC.02

塑料物流

来自模型店的流道 & 浇口

我们的塑料原料来自比例模型套件：零件剪下后剩余的流道和浇口。

SPEC / 塑料收料RL-P02
  • 来源：模型、拼装套件和游戏店
  • 材料：流道 & 浇口 · 聚苯乙烯 / ABS
  • 状态：洁净、干燥，天然不受污染
  • 分拣：尽可能按颜色分开
  • 市场：PS/ABS · BC 无买家付费；最近的在 Ontario2
  • 去向：破碎 → 分拣 → 干燥 → 注塑成型

每套模型套件（Gunpla 及其同类）都以固定在塑料框架上的零件形式到货。一旦拼装完成，这个框架就被丢弃。它是可获取的最洁净的废弃物流之一：均匀、高品质的聚苯乙烯和 ABS，按已知规格成型，按框架分拣，并且没有任何食物或液体污染。

模型和拼装店是理想的合作伙伴。他们的顾客稳定地产生这种确切的边角料，而且许多制作者早已把流道保存起来，而非丢弃。我们在店内放置一个收集箱；材料便直接流入我们的注塑成型闭环。

像这样的工业下脚料树脂比路边回收的塑料更洁净、更均匀，因此能获得溢价1，然而 BC 没有项目接收企业塑料，本地也无买家为 PS 或 ABS 付费2。这道缺口正是我们的原料。

SEC.03

电池物流

来自维修、电子烟 & 零售的电芯

凡有设备销售或维修之处，锂电芯便会积累。我们与这些商家合作，在中途截取它们。

SPEC / 电芯收料RL-H02
  • 来源：手机 / IT 维修店
  • 来源：电子烟零售商
  • 来源：电子产品零售商（如 Best Buy）
  • 材料：来自手机、一次性设备和其他设备的锂电芯
  • 处理：端子贴胶，存放于防火容器中
  • 运输：锂危险品 · UN3480 / UN3481，Class 93
  • 去向：Project HEX 回收 + 重建

报废电芯在路边收集箱中是危险品，但在合适的人手中却是宝藏。我们通过每天经手这些电芯的商家收集它们，并将其送入 Project HEX 进行测试、分级，重建为模块化电力单元。

三类合作伙伴供给这一物流：

手机 & IT 维修店：更换电池和维修主板时取下的电芯，否则便被囤积或丢弃。
电子烟零售商：用尽的一次性产品和模块，每个都含一枚通常在尚有余电时便被丢弃的锂电芯。
电子产品零售商：如 Best Buy 这样的大型商店，已在运营回收计划并大量收集电芯。

BC 目前没有电子烟回收计划；一项由生产者出资的计划已立法定于 2027 年 10 月实施，方案须在 2026 年 12 月前提交；尚未指定任何处理商，即便在 Quebec 已运行的计划中也是如此4。被丢弃的电芯还在 2025 年引发了横跨美国和加拿大的 448 起废弃物设施火灾5。现在就截取它们，既是一项安全措施，也是一步先手。

SEC.04

合作网络

谁托管收集箱

每家合作伙伴托管其顾客本就产生的物流。

TYPE 01模型 & 拼装店

模型套件和游戏店，顾客拼装完成后会留下剩余的框架。

投放：流道 & 浇口
TYPE 02手机 & IT 维修

维修台更换故障电池并积累取下的电芯，却没有洁净的出路。

投放：锂电芯
TYPE 03电子烟零售商

电子烟店收集用尽的一次性产品，每个内部都封着一枚锂电芯。

投放：一次性电芯
TYPE 04电子产品零售商

如 Best Buy 这样的大型商店，已建立回收计划并大量收集电芯。

投放：混合设备电芯
