SEC.01 問題 // 被完整丟棄的價值

我們丟棄了大部分壽命還在的材料。

6.5%
加拿大每年丟棄的 5,152 kt 塑膠中，僅約 6.5% 再處理為新材料；其餘皆被掩埋或焚燒。1
$0 in BC
沒有任何 BC 計畫收購工業後的 PS 或 ABS，也就是每套模型套件裡的樹脂。最近的付費買家約在 4,000 km 之東的 Ontario。2
Oct 2027
BC 由生產者出資的電子菸回收已立法卻尚未建置；即使在 Quebec 已上路的計畫中，也尚未指定任何處理商。4

廢棄物流不是一個處置問題。它是被錯誤定價的庫存：已分類、已濃縮，每天送達一組可預期的門口。

SEC.03 模式 // 在銷售點收集

我們無須四處尋找的供應。

我們在那些產品會變成廢棄物的企業設置投放桶；材料從它自己的前門回流，預先分類且乾淨。像這樣的工業後樹脂比路邊回收的塑膠更乾淨、更均質，因此享有溢價3；我們的則免費到手，因為今日 BC 沒有買家在爭搶 PS 或 ABS。2

SEC.04 產品線 // 兩股物流，一個循環

兩種原料。兩條產品線。

Plastic Works
模型套件的 PS/ABS 框架經破碎、分類，並射出成型為耐用製品；這些正是 BC 沒有買家的樹脂，2 在地再製、在地銷售。
Project HEX
回收的鋰電芯經測試、分級，並重建為模組化的六角形電力磚：旗艦產品。

Plastic Works 支撐近期營收與收集網路；Project HEX 則是創投規模的機會。

SEC.06 願景 // 一層自充電表皮

六角形能密鋪。於是我們包覆整棟住宅。

每片磚都發電、儲電並連接。扣合足夠多片，陣列便成為建築的外表面：一棟自給電力的測地住宅。陣列可無縫擴展、包覆複合曲面，並以一次一片回收磚的方式成長。廢棄物流化為分散式能源基礎設施。
