加拿大每年丟棄 5,152 kt 的塑膠，卻僅約 6.5%1 獲得再處理；電池與裝置在其大部分使用壽命仍在時就被丟進垃圾桶。Riposte Laboratories 的創立正是為了直接面對這個問題：從廢棄物流中回收材料，並將其化為耐用、精良工程的產品送回。

我們以在地規模運作：回收原本會被丟棄的材料，將其設計成實用產品，再送回它們原本所屬的社區。

SEC.02

Plastic Works

// 在地循環射出成型
SPEC / 塑膠循環RL-P01
  • 原料：乾淨的工業後 PS & ABS（模型套件框架）
  • 製程：破碎 → 分類 → 乾燥 → 射出成型
  • 產出：耐用的再利用製品，自家成型
  • 半徑：在地。就地收集，就地再製，就地使用
  • 原則：讓材料就近持續使用

塑膠從在地來源收集，再經破碎、分類，並透過射出成型重新製造。

工業後的 PS 與 ABS（每一套模型套件裡的樹脂）乾淨、單一樹脂且均質2，然而 BC 沒有任何計畫收取企業的塑膠，也沒有在地買家願意付費收購3。我們讓這個循環維持在地：社區規模的收集、自家處理，並將成型產品送回它們的來源地區。

短供應循環。長壽命產品。無需跨洋運輸。

SEC.03

Project HEX

// 模組化回收電力
SPEC / HEX 單元RL-H01
  • 電芯：6 × 圓柱形鋰離子，45 × 13 mm · 回收並分級
  • 拓撲：每邊一顆電芯，接成 6S（串聯）
  • 能量：6 × 2.41 Wh = 每片 14.46 Wh
  • 頂面：整合式光電面板 · 自充電
  • 邊緣：磁性 · 磚片扣合成密鋪陣列
  • 配件：磁吸螢幕，用於介面＋遙測
  • 幾何：六角形，可在測地結構上密鋪
  • 狀態：開發中 · 專利申請中

Project HEX 將回收的鋰電芯轉化為模組化的六角形電力磚：一片可扣接成陣列並包覆建築表面的能源面板。

每個單元搭載六顆尺寸為 45 × 13 mm 的圓柱形電芯，每邊各安置一顆於六角形之上，並以串聯接線。六顆回收的 2.41 Wh 電芯以 6S 輸出，每片提供 14.46 Wh。

頂面是太陽能板，因此每片磚都是一台為自身電芯補電的小型發電機。磁性邊緣讓單元彼此扣合，同樣的磁鐵也可接上一個選配的螢幕模組，用以與陣列連接。

由於六角形能密鋪，陣列可無縫擴展並包覆複合曲面；這種幾何是為了給以測地為基礎的住宅披上一層自充電電力而打造。
